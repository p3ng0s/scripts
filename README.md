# scripts

p3ng0s script dump used for p3ng0s daily usage.
All of the scripts should be installed with `755` permissions inside of `/usr/local/bin/`.

## Files:
 - `256-color.sh` : show all 256 colors
 - `backup.sh` : A simple script to backup files from your `$HOME`.
 - `banner.sh` : A script to show a random banner in a terminal.
 - `darthvader.sh` : Darth vader banner.
 - `dmenu_emoji` : A dmenu menu to show emojies.
 - `dmenu_keyboard` : A dmenu menu that collects a few X11 automation ideas.
 - `dmenu_ovpn` : A dmenu menu that allows you to connect to vpns.
 - `dmenu_shut` : A dmenu menu to shutdown a machine.
 - `dmenu_splain` : A dmenu menu to show off man pages.
 - `dmenu_win` : A dmenu menu to manage a qemu VM.
 - `extract.sh` : A script to extract compressed files.
 - `monitor.sh` : A script to monitor a network (deprecated).
 - `open` : Open stuff.
 - `pacman.sh` : A pacman banner script.
 - `wallpaper-gen` : A python script to generate a wallpaper from a collection of images.
 - `wp6.sh` : Wifi pineapple 6 routing script.
 - `wp7.sh` : Wifi pineapple 7 routing script.
 - `xurls` : Find urls in a X window.
 - `ytm_extractor.py` : Extract images from a youtube music webpage network traffic dump.
